View In Browser

View In Browser is a Sublime Text 2 plugin that will open whatever is in your current view/tab in your browser of choice. If the file you are working with is new and has not been saved then this plugin will save a temporary copy fo you and then open it up. View In Browser also works with local application servers to open your files using a URL prefix.
